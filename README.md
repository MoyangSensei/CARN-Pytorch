# CARN-PyTorch

This repository is an official PyTorch implementation of the paper **"A Super-Resolution Network Using Channel Attention Retention for Pathology Images"**.

## Requirement
* torch == 1.8.0
* python >=3.6.0
* cuda == 11.1

Other see `requirement.txt`
